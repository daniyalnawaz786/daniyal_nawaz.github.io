Supermarket Sales Prediction

End-to-end machine learning pipeline that predicts supermarket branch sales from historical transaction data. The project covers the full data science workflow: data cleaning, exploratory analysis, feature engineering, model selection and training with Scikit-learn, and performance evaluation using metrics like RMSE and R². Turns raw sales records into accurate revenue forecasts that support smarter inventory planning and business decisions.

🎯 Predictive regression model on real sales dataset
🔧 Full pipeline: cleaning → EDA → features → train → evaluate
📊 Visualizations of patterns, trends & model performance
